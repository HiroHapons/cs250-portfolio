## About This Repository

This repository contains artifacts from CS 250: Software Development Lifecycle, completed at Southern New Hampshire University. Throughout this course, I took on various roles within a Scrum-Agile team while developing the SNHU Travel application for a fictional client. The work here reflects my experience applying Agile principles across the full software development lifecycle.

## Reflection

This course gave me hands-on experience with the Scrum-Agile framework by rotating through each team role across the semester. The most valuable lesson was seeing how Agile's iterative structure allows a team to absorb mid-sprint changes, like the shift to wellness travel in Module Five, without losing momentum. As someone pursuing a career in data science and bioinformatics, understanding how development teams operate in Agile environments will be essential for collaborating on software projects in technical workplaces.
